# User Accounts
## To know the current user
```
whoami
```

### To know the id
```
id
```


## To switch user
```
su ${username}
```


# Download Files

## To download a file and save it locally
```
wget ${url_of_file} -O ${new_file_name}
curl wget ${url_of_file} -O
```


## To check OS version
```
ls /etc/*release*
```


## To know more information of the OS
```
cat /etc/*release*
```
